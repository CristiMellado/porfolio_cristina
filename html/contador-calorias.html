Contador de Calorías y Ejercicio

Aplicación web desarrollada con React, Vite y Tailwind CSS que permite registrar comidas y ejercicios para calcular el balance total de calorías consumidas y quemadas. Implementa gestión de estado mediante useReducer, almacenamiento persistente con localStorage y una interfaz optimizada para ofrecer una experiencia fluida e intuitiva.
