Vivero

Inventario, ventas y clientes para un vivero de plantas

Resumen

Vivero es una aplicación web fullstack para la gestión integral de un vivero de plantas. Permite administrar inventario, registrar ventas y gestionar clientes con una interfaz clara y responsive.

Funcionalidades clave

  • • Catálogo e inventario de plantas con stock y categorías.
  • • Gestión de ventas y tickets básicos.
  • • Fichas de clientes y búsquedas rápidas.
  • • Informes sencillos (productos más vendidos, stock bajo).

Desafíos y soluciones

  • • Modelado de entidades (Producto, Venta, Cliente) → diseño claro del dominio y relaciones.
  • • Controladores y rutas → Spring MVC con mapeos REST y separación de capas (controller/service/repository).
  • • Vistas server-side → plantillas Thymeleaf con fragmentos reutilizables para listas y formularios.
  • • Validación de datos → Bean Validation en formularios con mensajes de error amigables.
  • • Interacciones de UI → JavaScript para validaciones y mejoras de usabilidad.

Resultados y aprendizajes

Fortalecí el diseño de dominio y el patrón MVC con Spring, la construcción de vistas con Thymeleaf y la validación en servidor. En el frontend, consolidé HTML accesible, CSS responsive y JS para interacciones ligeras que facilitan el flujo de trabajo del día a día en el vivero.
