Volavia

Volavia es una aplicación web fullstack que ayuda a organizar viajes de forma visual y sencilla. Permite crear itinerarios, registrar anécdotas y controlar los gastos, ofreciendo una experiencia clara, personalizable y atractiva.

Funcionalidades clave

  • Seleccionar un país y organizar el viaje a partir de él.
  • Diario de viaje con fotos y recuerdos.
  • Control de gastos por categorías y totales.
  • Registro de anécdotas por destino.

Desafíos y soluciones

  • Renderizado en servidor con Thymeleaf usando fragmentos y layouts reutilizables.
  • Arquitectura MVC con Spring y mapeos REST organizados por responsabilidad.
  • Validación de formularios y seguridad con Bean Validation y CSRF.
  • Interactividad ligera con JavaScript para mejorar la experiencia de usuario.
  • Diseño responsive y coherente con HTML semántico y CSS modular.

Resultados y aprendizajes

Aprendí a consolidar el patrón MVC con Spring, la composición de vistas con Thymeleaf (layouts y fragmentos) y la validación de datos en el servidor. En frontend, reforcé mis conocimientos en HTML semántico, CSS responsive y pequeñas interacciones JavaScript para mejorar la UX.
