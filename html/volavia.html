Resumen

Volavia es una aplicación web fullstack que ayuda a organizar viajes de forma visual y sencilla. Permite crear itinerarios, registrar anécdotas y llevar el control de los gastos, priorizando una experiencia clara y agradable.

Funcionalidades clave

  • • Seleccionar un país y apartir de ahí organizar su viaje
  • • Diario de viaje con fotos y recuerdos.
  • • Control de gastos por categorías y totales.
  • • Añadir anécdotas en cada destino

Desafíos y soluciones

  • • Renderizado del lado servidor con plantillas → uso de Thymeleaf con modelos, fragmentos y layouts reutilizables.
  • • Organización de controladores y rutas → Spring MVC con mapeos claros (REST) y separación por responsabilidades.
  • • Validación y seguridad en formularios → Bean Validation (JSR-380) y protección CSRF de Spring Security.
  • • Interacciones dinámicas en la UI → JavaScript para mejorar UX (validaciones, navegación suave y pequeñas mejoras).
  • • Diseño responsive y consistencia visual → HTML semántico y CSS modular siguiendo la paleta del proyecto.

Resultados y aprendizajes

Consolidé el patrón MVC con Spring, la composición de vistas con Thymeleaf (layouts y fragmentos), y la validación de datos en el servidor mediante Bean Validation. A nivel de frontend, mejoré la estructura y accesibilidad del HTML, el uso de CSS para un diseño responsive y el JavaScript necesario para interacciones ligeras que complementan el renderizado en servidor.
